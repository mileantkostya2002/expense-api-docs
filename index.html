Complete API documentation for expense tracking system

üîê Authentication

All endpoints (except public ones) require Bearer token authentication:

Authorization: Bearer YOUR_API_TOKEN

üí∞ Expenses

GET /api/v1/expenses

Get list of expenses for authenticated user (current month by default)

Query Parameters:
limit=20            # Results per page
offset=0            # Pagination offset
state=draft         # Filter by state
date_from=2025-10-01
date_to=2025-10-31
q=taxi              # Search in name/description
Example Response:
{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": "Taxi to meeting",
      "total_amount_currency": 45.50,
      "date": "2025-10-06",
      "state": "draft"
    }
  ],
  "pagination": {
    "total_count": 15,
    "limit": 20,
    "offset": 0
  }
}
POST /api/v1/expense

Create new expense

Request Body:
{
  "name": "Taxi to client meeting",
  "total_amount_currency": 45.50,
  "product_id": 12,
  "date": "2025-10-06",
  "description": "Transportation expense",
  "payment_mode": "own_account"
}
Response:
{
  "success": true,
  "expense_id": 25,
  "message": "Expense created successfully"
}
GET /api/v1/expense/{expense_id}

Get single expense details

Example:
GET /api/v1/expense/25
Response:
{
  "success": true,
  "data": {
    "id": 25,
    "name": "Taxi to meeting",
    "total_amount_currency": 45.50,
    "state": "draft",
    "attachments": []
  }
}
PUT /api/v1/expense/{expense_id}

Update expense (only in draft/reported state)

Request Body:
{
  "name": "Updated expense name",
  "total_amount_currency": 50.00,
  "date": "2025-10-07"
}
DELETE /api/v1/expense/{expense_id}/delete

Delete expense (only in draft/reported state)

Response:
{
  "success": true,
  "message": "Expense deleted successfully"
}

üìã Reports

GET /api/v1/expense/reports

Get list of expense reports

Response:
{
  "success": true,
  "reports": [
    {
      "id": 5,
      "name": "October Expenses",
      "state": "submit",
      "total_amount": 350.00,
      "expense_count": 7
    }
  ]
}
POST /api/v1/expense/create-report

Create expense report from multiple expenses

Request Body:
{
  "name": "Monthly Expenses October",
  "expense_ids": [1, 2, 3, 4]
}
Response:
{
  "success": true,
  "report_id": 6,
  "report_name": "Monthly Expenses October",
  "expense_count": 4
}
GET /api/v1/report/{report_id}

Get report details with all expenses

POST /api/v1/report/{report_id}/submit

Submit report for approval

Response:
{
  "success": true,
  "message": "Expense report submitted successfully",
  "new_state": "submit"
}

üìé Attachments

GET /api/v1/expense/{expense_id}/attachments

Get list of attachments for expense

Response:
{
  "success": true,
  "attachments": [
    {
      "id": 10,
      "name": "receipt.pdf",
      "mimetype": "application/pdf",
      "file_size": 15420
    }
  ]
}
POST /api/v1/expense/{expense_id}/attachments

Upload attachment (base64 encoded)

Request Body:
{
  "file_name": "receipt.pdf",
  "file_data": "JVBERi0xLjQKJeLjz9MKMSAwIG9iago8..."
}
GET /api/v1/expense/file/{expense_id}/attachments/{attachment_id}

Download attachment file (returns binary data)

DELETE /api/v1/expense/{expense_id}/attachment/{attachment_id}

Delete attachment
